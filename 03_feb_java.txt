Que 4-what is a string constant pool ?
Ans 4-
The string constant pool is a special area in the Java heap memory that is used to store a pool of unique string objects. In other words, it is a cache-like mechanism used to optimize memory usage for strings in Java.

When a string is created using a string literal (e.g., "Hello"), Java checks if the string already exists in the string constant pool. If it does, the reference to the existing string object is returned instead of creating a new one. This means that multiple references to the same string literal will point to the same object in the string constant pool, saving memory.
